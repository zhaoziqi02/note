#上传文件：

    git init  //初始化本地仓库

    git add .   //从编辑区添加到暂存区

    git commit -m '命名'  //暂存区提交到分支

    git remote add origin git@github.com:zhaoziqi02/note.git   //连接


    git push -u origin master  //同步远程仓库



    SHIFT+ALT+F      //整理编辑器里代码
